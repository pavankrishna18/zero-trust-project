ℹ️ IP Whitelist Help

Formats Accepted:

  • 192.168.1.100 - Single IP address
  • 192.168.1.0/24 - IP range (CIDR notation)
  • 10.0.0.0/8 - Large network range
  • 127.0.0.1 - Localhost (always recommended)

Common Examples:

  • Office Network: 192.168.1.0/24
  • VPN Range: 10.8.0.0/24
  • Single Remote Worker: 203.0.113.45
  • Localhost: 127.0.0.1

⚠️ Important Notes:

  • Admins can bypass IP whitelist (configurable)
  • Always add 127.0.0.1 for local access
  • Inactive IPs are stored but not checked
  • Changes take effect immediately

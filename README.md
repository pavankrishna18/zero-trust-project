# ZeroTrustX - Zero Trust Network Access Framework

A comprehensive ZTNA implementation using Python Flask with multi-factor authentication, device fingerprinting, role-based access control, and real-time monitoring.

## Features

- **Multi-Factor Authentication**: OTP-based 2FA using TOTP
- **Device Trust Evaluation**: Fingerprinting and trust management
- **Role-Based Access Control**: Granular access policies
- **Real-Time Monitoring**: Activity logging and anomaly detection
- **Session Management**: JWT-based secure sessions
- **Admin Dashboard**: Comprehensive security management
